# ComfyUI PromptPalette

## プロジェクト概要

ComfyUI-PromptPaletteは、マウス操作で素早いプロンプト編集を可能にするComfyUI向けのカスタムノードです。
チェックボックスによるフレーズの有効/無効切り替えや、ウェイト調整ボタンといった、インタラクティブなプロンプト編集機能を提供します。

## 開発環境

- ComfyUI
  - Node 1.0対応
- Python
- Node.js

## コーディング規約

- コードコメントは英語
- ログ出力は英語

## コミット規約

- コミットメッセージは英語

## インストール方法

標準的な ComfyUI カスタムノードの導入手順に従い、`custom_nodes` ディレクトリへ配置する。
追加のセットアップや依存関係は不要。

## プロジェクト構成

ComfyUIのカスタムノード構成に従う。

- `__init__.py`: ノードのマッピングと Web ディレクトリをインポート/エクスポートするエントリポイント
- `nodes.py`: テキスト入力を処理する `PromptPalette` クラスを含むバックエンド（Python）
- `web/index.js`: ComfyUI の app システムに登録し、カスタム UI を提供するフロントエンド（JavaScript）
- `pyproject.toml`: Comfy Registry 公開のためのメタデータ

## 機能

### ノードの入出力

- 複数行テキスト入力を受け取り、空行と `//` で始まるコメント行を除外
- 行内の `//` 以降はインラインコメントとして除去
- コメントでない行の末尾にカンマ区切りを付与（既にカンマがあれば追加しない）
- `prefix` 入力があれば先頭に付与（改行で結合）
- 整形済み文字列を出力

### 編集モードと表示モードの切り替え

- ノード下部の `Edit` / `Save` ボタンで編集モードと表示モードをトグル
- デフォルトは表示モード
- ノード作成時にテキストウィジェットを非表示化し、`Edit` ボタンを追加

### 編集モード時のUI

- 標準の複数行テキストウィジェットと `Save` ボタンを表示
- 表示モードのキャンバス描画（チェックボックス等）は行わない

### 表示モード時のUI

- 表示モードではキャンバス描画の UI を表示
- 行ごとにチェックボックス、フレーズテキスト、ウェイト表示、± ボタンを描画
- 有効/無効で色分けし、ウェイトが 1.0 以外の行は太字表示
- ウェイト表示は 1.0 のとき非表示
- フレーズ表示はウェイト表記と末尾カンマを除去したテキストを用いる
- テキストが空の場合はキャンバス中央に "No Text" と表示
- 行数に応じてノード高を自動調整
- ノードが折りたたまれている場合は描画をスキップ
- クリック処理は座標ベースの `clickableAreas` により判定
- クリック判定用の `clickableAreas` は描画ごとに再生成

### チェックボックスによるフレーズの有効/無効切り替え

- チェックボックスクリックで行頭の `//` コメントをトグル
- 行頭コメントとインラインコメントを考慮して表示テキストを抽出
- インラインコメントは表示テキストから除外しない

### ウェイト調整

- `(phrase:1.5)` のようなウェイト表記を解析し、1.0 以外のウェイトを表示
- ± ボタンで 0.1 刻みの増減
- ウェイトの範囲は 0.1〜2.0
- 末尾カンマを除去してからウェイト表記を再構成
- 1.0 に戻すとウェイト表記自体を外して元のテキストに戻す
- 上限超過の値は増加を拒否し、減少時のみ上限にクランプ
- 行頭コメント行やインラインコメント付き行でも増減可能
- コメントのみの行（`//` のみ）はウェイト操作をスキップ

### テーマ連携

- ComfyUI の CSS 変数から色を取得し、描画に利用
- ライト/ダーク両テーマに対応
- パフォーマンスのためのカラーキャッシュ
- 3 桁 16 進カラーの展開に対応

### 拡張登録

- ComfyUI 拡張として登録し、PromptPalette ノードの生成/描画にフック
- `beforeRegisterNodeDef` で PromptPalette ノードの挙動を差し替え

## 編集ポリシー

- 要求にない変更はせず、指定された箇所のみ最小限の変更に留める。
- 要求に不明な点や曖昧な点があれば確認する。勝手な推測をしない。
- コードを編集する際は、その変更意図を説明する。
- チャットでの回答は日本語。
